Components

This page demos a set of web components created with Lit. The rendered components are followed by their corresponding source code.

Lit is a small library that makes building web components easier - providing support for reactive properties, simpler templating syntax, and more. These components were used to vet the technology, to uncover benefits and drawbacks, and are not proposed as-is for production.

The Internet Archive UX team has adopted Lit and is introducing web components to the site.
